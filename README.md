# Databases
This project is a Travel Agency Management System created for a Database Laboratory course. 
The goal was to design a database to manage branches, employees, trips, and reservations. We extended the system to include features like an IT admin log, travel offers, and performance testing with over 60,000 records.

### Technologies Used
MySQL: Used to create the database, tables, and handle data storage.

SQL (Stored Procedures & Triggers)

Java: Used to build a desktop application (GUI) so users can easily interact with the data.

JDBC: Used to connect the Java application to the MySQL database.
